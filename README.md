# Applied Statistics Master's Degree (Maestría en Estadística Aplicada)

This repository contains examples and scripts from my master's degree classes in Applied Statistics.

## Directory Structure

The repository is organized by subject:

-   **mep**: *Modelos Estadísticos Paramétricos* (Parametric Statistical Models)
    -   Contains scripts for statistical distributions (Normal, T, Sampling distributions) and hypothesis testing.
-   **mrl**: *Modelos de Regresión Lineal* (Linear Regression Models)
    -   Contains class examples and regression models.
-   **se**: *Software Estadístico* (Statistical Software)
    -   Contains data manipulation examples, file I/O operations, and general R programming exercises.

## Usage

Scripts can be run using R or RStudio. See `AGENTS.md` for more technical details and guidelines regarding code style and project structure.
